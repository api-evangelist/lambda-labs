Lambda Cloud launched workspaces, logical containers that help teams organize GPU resources, control access, and separate development, staging, and production environments in shared infrastructure. Each account starts with one default workspace and admins can create up to 200 additional workspaces at no extra cost, with GPU instances, Kubernetes clusters, and filesystems all residing within workspaces as the primary unit for access control.
